# 📱 Calculadora App

Calculadora mobile desenvolvida com **React Native + Expo**, com suporte a modo claro e escuro.

## ✨ Funcionalidades

- Operações básicas: adição, subtração, multiplicação e divisão
- Porcentagem e inversão de sinal
- Modo claro e escuro com toggle
- Exibição da expressão em tempo real
- Interface responsiva e moderna
- Vibração haptica nos botões (Android)

## 🛠️ Tecnologias utilizadas

- [React Native](https://reactnative.dev/)
- [Expo](https://expo.dev/)
- JavaScript

## 🚀 Como rodar o projeto

**Pré-requisitos:** Node.js instalado

    git clone https://github.com/JhonatasSantos/calculadora-app.git
    cd calculadora-app
    npm install
    npx expo start --web
